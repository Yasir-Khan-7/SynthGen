Run SynthGen AI Locally

SynthGen AI is a Streamlit application that requires Python to run. Follow these steps to run it on your local machine:

  1. Clone the repository:
    git clone https://github.com/Yasir-Khan-7/SynthGen.git
  2. Navigate to the project directory:
    cd SynthGen
  3. Install the required dependencies:
    pip install -r requirements.txt
  4. Run the Streamlit app:
    streamlit run generator.py
